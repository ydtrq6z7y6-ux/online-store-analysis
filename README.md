# Online Store Data Analysis

This project demonstrates basic data analysis using Python and Pandas on an online store dataset.

##  Features

* Total number of products
* Most sold product
* Top 5 smartphones by sales
* Laptop price analysis (min & max)

## Technologies

* Python
* Pandas

## How to run

```bash
pip install pandas
python task1_online_store_analysis.py
```

##  Dataset

The dataset includes:

* product name
* category
* price
* quantity sold
* ratings
